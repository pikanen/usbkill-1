what is Usb-kill?
usbkill 은 anti-forensic 적재 가능 커널 모듈 ( Loadable kernel module , LKM ) kill-switch로서 당신의 usb포트의 변화를 기다리고 변화가 생기면 컴퓨터를 종료시킨다.
헤더파일 분석 : 임세준
메인.c 분석 : 정세영, 김주현, 정재헌, Piao Renyu
새로운 기능 제안 : 
1)컴퓨터가 실행될때마다 모듈을 새로 시작해 주어야 하므로 자동실행 기능 추가
2)본인이 컴퓨터를 실행하고있고 usb를 사용할 일이 있을때마다 모듈을 꺼주기가 귀찮으므로 암호 기능을 만들어 정해진 시간내에 암호를 입력하면 꺼지지않는 기능 추가
